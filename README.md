# job_scrapper #

## 설명

- Python과 Flask를 이용하여 만든 잡 스크래퍼
- 검색 창에 개발 언어/프레임워크를 입력한 후 'Search' 버튼을 누르면 'Stack overflow'로부터 직업 스크래이핑 시작
- 검색한 결과를 웹 페이지에 그리드로 보여줌
- 'Export to CSV' 버튼을 누르면 CSV 파일 다운로드
